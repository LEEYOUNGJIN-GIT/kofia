## 표준코드 확정 절차

1. [dis.kofia.or.kr](http://dis.kofia.or.kr) 또는 (선택) 공공데이터 `getStandardCodeInfo`로 검색
2. **클래스(C-e / C-E / Ce 등)** 가 보유 클래스와 일치하는지 확인
3. 12자리 `srtn_cd`를 본 표·`fund_list.yaml`에 기록
4. `alias`는 영문·짧은 식별자 (예: `TDF2050_Ce`) — 파일명·CSV 조인용
